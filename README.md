## Features

<div align=center><img src="https://raw.githubusercontent.com/IAFEnvoy/MineDash/refs/heads/master/img/2.webp" style="width:400px;text-align:center;" alt=""></img></div>

### Added

- Game Play entity (Click to bind controls, use arrow keys to move.)
- Various building blocks (Some of them have builtin connected textures.)
- Spikes
- Rings & Pads
- Hitbox Display (Open in config)

### Planned

- Classic mode & First person mode
- More play modes
- Portals
- Music system
- Custom model loader
